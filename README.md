# 📞 Contact Us Form - React Project
A responsive and interactive Contact Us page built using React.js. This project demonstrates form handling, reusable components, and clean UI design.
##  Features

- 📩 Contact form with Name, Email, and Message fields
- 🔄 Real-time state updates using React Hooks
- 🧩 Reusable Button component
- 🎨 Styled using CSS Modules
- 📞 Multiple contact options (Chat, Call, Email)
- 🖼️ Responsive layout with image section
- ⚡ Form submission handling

## 🧠 Concepts Used

This project helped me understand:

- React Functional Components
- useState Hook (Form Handling)
- Controlled & Uncontrolled Components
- Event Handling (onSubmit)
- Component Reusability
- CSS Modules (Scoped Styling)
- Props & Conditional Styling


## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS Modules
- React Icons
